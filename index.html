Individual Projects

  • Productivity Aid- Google Chrome Extension:

    App to block website hosts to encourage productivity! Dynamically stores and allows users to easily add and remove hosts that they would like to be blocked. Users can toggle the blocking mode and the app stores/syncs the mode and blocked host list to chrome storage. Closes all tabs under a blocked host and redirects user to “blocked” page when blocking is enabled.

  • Chatting Application/Discord Clone:

    Uses StreamChat API to create channels for messaging text, images, audio, and video. Amazon AWS API allows for unique user sign in and account creation using email. Hosts servers that allowed multiple channels with dynamic and interactive member lists, like Discord, and direct messaging. Users can toggle between servers and direct messages, create new servers, create new channels within servers, and start new direct messages.

  • Pomodoro-Chrome extension (in development):

    Extended version of Productivity Aid. Allows users to set custom times for working and breaking and cycles between the two for studying in the Pomodoro method
